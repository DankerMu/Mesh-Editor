# 14. 编辑引擎详细设计

## 14.1 设计目标

编辑引擎负责把前端绘制的区域和编辑参数转化为对 `qpf_after`、`ptype_after` 的确定性修改，并生成统计、mask、操作日志。编辑引擎必须满足：

```text
preview 与 apply 一致
所有操作可撤销/重做
不修改缺测和负值异常格点
qpf 与 ptype 保持基础一致性
所有操作可复盘
```

---

## 14.2 输入与输出

### 输入

@dataclass
class EditContext:

## 14.3 valid_mask

`valid_mask` 是所有编辑操作的基础约束。

```text
valid_mask = not missing_mask and not negative_qpf_mask
```

任何编辑操作只能作用于：

```text
operation_mask & valid_mask
```

缺测、负值异常格点不参与编辑、不参与统计、不进入 affected_grid_count。

---

## 14.4 Mask 生成

### 14.4.1 多边形 mask

输入：

```json
{"type": "polygon", "coordinates": [[lon1, lat1], [lon2, lat2], ...]}
```

算法：

```text
1. 校验 polygon 至少 3 个点；
2. 经纬度钳位或拒绝越界点；
3. 使用 shapely Polygon；
4. 对格点中心点执行 contains/intersects；
5. 返回 bool[501,821]。
```

### 14.4.2 绘制笔 line_buffer mask

输入：

```json
{"type": "line_buffer", "width_grid": 2, "coordinates": [[lon1, lat1], ...]}
```

算法：

```text
1. 将 width_grid 转为经纬度近似缓冲宽度；
2. 使用 LineString.buffer；
3. 转 polygon mask。
```

### 14.4.3 笔刷 brush_path mask

输入：

```json
{"type": "brush_path", "radius_grid": 3, "points": [[lon1, lat1], ...]}
```

算法：

```text
1. 每个点转换为 grid_i/grid_j；
2. 以 radius_grid 生成圆形格点邻域；
3. 所有邻域取并集；
4. 返回 bool mask。
```

## 14.5 降水编辑操作

### 14.5.1 定值

```python
def apply_qpf_set_value(ctx, mask, value) -> EditResult:
    target = mask & ctx.valid_mask
    qpf_new = ctx.qpf_current.copy()
    ptype_new = ctx.ptype_current.copy()
    qpf_new[target] = max(value, 0)
    ptype_new = enforce_qpf_ptype_consistency(qpf_new, ptype_new, ctx.qpf_threshold)
    return build_result(...)
```

约束：

```text
value >= 0
如果 value <= threshold，则 ptype 自动置 0
如果 value > threshold 且原 ptype=0，前端应要求用户选择目标 ptype，或默认保持 ptype=0 并触发提示
```

### 14.5.2 增加

```python
qpf_new[target] = qpf_current[target] + delta
```

约束：

```text
delta > 0
only_nonzero=true 时 target &= qpf_current > threshold
```

### 14.5.3 减少

```python
qpf_new[target] = np.maximum(qpf_current[target] - delta, 0)
```

减少后自动执行一致性：

```text
qpf <= threshold → ptype=0
```

### 14.5.4 乘系数

```python
qpf_new[target] = qpf_current[target] * factor
```

约束：

```text
factor >= 0
factor = 0 等价于清零
```

### 14.5.5 清零

```python
qpf_new[target] = 0
ptype_new[target] = 0
```

## 14.6 相态编辑操作

### 14.6.1 设置相态

```python
def apply_ptype_set(ctx, mask, target_ptype) -> EditResult:
    target = mask & ctx.valid_mask & (ctx.qpf_current > ctx.qpf_threshold)
    ptype_new = ctx.ptype_current.copy()
    ptype_new[target] = target_ptype
    ptype_new[ctx.qpf_current <= ctx.qpf_threshold] = 0
```

约束：

```text
target_ptype in [0,1,2,3]
如果 target_ptype=0，则 qpf 是否同步清零由 operation 决定：
  - set_ptype_zero_only：仅相态置 0，若 qpf>threshold 触发提示
  - clear：qpf 与 ptype 同步清零
推荐前端“设为无降水”使用 clear。
```

## 14.7 筛查清零

```python
def apply_screen_clear(ctx, selected_mask, threshold) -> EditResult:
    target = selected_mask & ctx.valid_mask & (ctx.qpf_current <= threshold)
    qpf_new[target] = 0
    ptype_new[target] = 0
```

`selected_mask` 可为当前选区，也可为全窗口。

---

## 14.8 一致性规则

硬规则：

```python
def enforce_qpf_ptype_consistency(qpf, ptype, threshold):
    ptype[qpf <= threshold] = 0
    qpf[qpf < 0] = 0
    ptype[~np.isin(ptype, [0,1,2,3])] = 0
    return ptype
```

保存前必须检查：

| 检查 | 处理 |
|---|---|
| qpf <= threshold 且 ptype != 0 | 自动修正 |
| qpf > threshold 且 ptype = 0 | 提示，允许用户选择清零或设置 ptype |
| ptype 非 0/1/2/3 | 自动置 0 并记录 warning |
| 作用于 invalid mask | 不允许，返回 affected_grid_count 不包含 invalid |

## 14.10 preview/apply 一致性

流程：

```text
preview:
  1. 读取 session_state
  2. mask_geometry → mask_raster
  3. 执行编辑操作
  4. 生成 preview_id
  5. 写 tmp/previews/{preview_id}.npz
  6. 返回统计和 preview_image

apply:
  1. 校验 preview_id 属于 session
  2. 读取 preview npz
  3. 不重新计算 mask 和结果
  4. 覆盖 session_state
  5. 写 edit_operation
  6. 删除 preview 缓存
```

---

## 14.11 undo/redo

推荐实现：

```text
undo/redo 不存反向 patch，而是从 base_version 重放操作。
```

流程：

```text
1. 从 session.base_version 读取 qpf/ptype
2. 按 sequence_no 顺序重放 is_undone=0 的操作
3. 得到当前 session_state
```

优点：

```text
- 逻辑简单；
- 避免 patch 累积误差；
- 保存版本时只需归档生效操作。
```

性能：501×821 网格较小，重放几十步可接受。后续可优化为快照。

## 14.13 编辑引擎测试点

```text
1. polygon mask 是否正确覆盖格点；
2. brush 半径是否对应格点数；
3. qpf 减少后不出现负值；
4. clear 后 ptype 同步为 0；
5. ptype 设置不影响 qpf；
6. qpf<=0.1 的格点强制 ptype=0；
7. preview 与 apply 结果完全一致；
8. undo 后数组回到上一状态；
9. redo 后数组恢复；
10. invalid mask 不参与编辑和统计。
```
