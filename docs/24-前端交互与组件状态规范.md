# 24. 前端交互与组件状态规范

## 24.1 文档定位

本文定义按钮启禁用、弹窗、状态卡、loading/empty/error、权限、preview/apply、字段加载、绘图任务等交互状态。目标是保证不同页面的行为一致，避免开发人员自行判断。

---

## 24.2 通用状态模型

每个业务组件至少支持以下状态：

| 状态 | 说明 | UI 表现 |
|---|---|---|
| loading | 请求中 | 骨架屏或 spinner |
| ready | 数据已加载 | 正常展示 |
| empty | 无数据 | Empty 组件 + 操作引导 |
| error | 加载失败 | 错误卡 + trace_id + 重试 |
| disabled | 无权限或不可操作 | 禁用按钮 + tooltip |
| partial | 数据部分缺失 | warning tag + 说明 |

---

## 24.3 按钮启禁用规则

### 24.3.1 编辑页按钮

| 按钮 | 启用条件 | 禁用原因示例 |
|---|---|---|
| 进入编辑 | `window.status=available` 或允许编辑的 partial | invalid、archived、locked |
| 预览 | session=editing 且存在有效 mask 且参数合法 | 未选择区域、参数无效 |
| 应用预览 | `preview_id` 存在且未过期 | 未预览、preview 过期 |
| 撤销 | `canUndo=true` | 无可撤销操作 |
| 重做 | `canRedo=true` | 无可重做操作 |
| 保存草稿 | session=editing 且 dirty=true | 无变更 |
| 提交审核 | version.status=draft | 非草稿状态 |
| 清空选区 | currentMaskGeometry 非空 | 无选区 |

### 24.3.2 审核页按钮

| 按钮 | 启用条件 |
|---|---|
| 审核通过 | version.status=submitted 且有 reviewer/admin 权限 |
| 退回修改 | version.status=submitted 且填写退回意见 |
| 保存审核意见 | comment 有变更 |
| 查看操作日志 | version_id 存在 |
| 生成复盘图 | version.status in [approved, released, submitted] |

### 24.3.3 复盘页按钮

| 按钮 | 启用条件 |
|---|---|
| 重新生成复盘图 | plot_status 非 running |
| 导出复盘包 | plot_status in [success, partial_success] |
| 下载图片 | image_path 存在 |
| 查看 manifest | manifest_path 存在 |
| 重试失败任务 | plot_status in [failed, permanently_failed] |

## 24.4 WindowSelector 交互

1. 选择起报时间后自动查询窗口。
2. 累计长度切换后只过滤窗口，不重新扫描数据。
3. `available` 行显示主按钮“进入编辑”。
4. `partial` 行显示“进入编辑”但带 warning tooltip。
5. `invalid` 行显示“不可编辑”。
6. 选中行时右侧详情同步显示。
7. 表格可按状态、累计长度、起始时效排序。

## 24.5 字段二进制加载交互

字段加载流程：

```text
loadSession → 返回 field_urls → 并行下载 qpf_after / ptype_after / invalid_mask → 初始化 PrecipPhaseGridLayer
```

加载状态：

```text
loadingFields=true 时地图显示骨架或浅灰 loading overlay。
字段下载失败时显示错误卡：字段加载失败，请重试（trace_id）。
```

前端校验：

```text
rows * cols * dtype_size 必须等于 arrayBuffer.byteLength。
不一致时抛 FIELD_BINARY_SIZE_MISMATCH（前端内部错误），不渲染图层。
```

## 24.6 DrawTools 交互

### polygon

```text
左键添加点；
双击完成；
Esc 取消；
Backspace 删除上一点；
完成后显示节点 handle，可拖拽修改；
至少 3 个点。
```

### line_buffer

```text
左键连续绘制线；
双击完成；
右侧可设置 width_grid；
地图显示线和缓冲区半透明面。
```

### brush_path

```text
按住左键绘制；
松开后形成一次 brush_path geometry；
radius_grid 可用滑块/输入框调整；
支持显示笔刷圆形预览。
```

## 24.7 preview / apply 交互

### 24.7.1 普通预览

```text
用户绘制 mask → 输入参数 → 点击预览 → 显示 preview overlay → 右侧展示统计 → 应用预览按钮启用。
```

### 24.7.2 取消预览

取消预览后：

```text
- 清除 preview overlay
- 清除 preview_id
- 保留当前 mask 和输入参数
- 不写 edit_operation
```

### 24.7.3 应用预览

应用成功后：

```text
- 更新 qpf_after / ptype_after 图层
- 清除 preview_id
- 清除 preview overlay
- OperationHistory 增加一条记录
- dirty=true
```

### 24.7.4 新增降水落区相态选择弹窗

触发条件：

```text
preview 返回 new_precip_needs_ptype=true
```

弹窗内容：

```text
标题：新增降水落区相态选择
说明：本次操作将新增 {new_precip_count} 个降水格点，请选择这些格点的相态。
选项：雨 / 雪 / 雨夹雪
按钮：取消 / 确认并应用
```

确认后 apply 请求携带 `target_ptype`。

## 24.8 OperationHistory 交互

| 操作 | 行为 |
|---|---|
| 点击行 | 打开详情 drawer，显示参数、mask、统计、op_ptype_transition |
| 撤销 | 标记 is_undone=1，重放操作，更新地图 |
| 重做 | 标记 is_undone=0，重放操作，更新地图 |
| 查看 mask | 地图高亮该操作 mask |

已撤销操作用灰色文字和“已撤销”标签显示。

---

## 24.9 审核页交互

1. 中央区域默认展示 before/after 双图对比（左右并排）。
2. 底部展示 6 张缩略图并排：before、after、delta_qpf、change_ptype、touched_mask、changed_mask。
3. 点击任一缩略图可在中央区域放大查看。
4. 退回修改必须填写意见。
5. 审核通过前可保存审核意见。
6. 审核通过后按钮置灰，状态变 approved。

---

## 24.10 复盘中心交互

1. 左侧目录选择 review 后，中央图片和右侧信息同步。
2. `success` 显示全部图片。
3. `partial_success` 显示 missing_fields 和占位图。
4. `failed` 显示失败原因、traceback 下载、重试按钮。
5. 导出复盘包为异步操作，显示任务进度。
6. 查看 manifest 使用 Drawer 或新标签页 JSON Viewer。

---

## 24.11 Loading / Empty / Error 文案

| 场景 | 文案 |
|---|---|
| 窗口为空 | 当前起报暂无可用窗口，请检查数据接入状态。 |
| 字段加载失败 | 字段数据加载失败，请重试。 |
| session locked | 当前窗口正在被其他用户编辑，请稍后重试。 |
| preview expired | 预览结果已过期，请重新预览。 |
| plot failed | 复盘图生成失败，请查看日志或重试。 |
| permission denied | 当前角色无权执行此操作。 |

---

## 24.12 权限展示

| 无权限行为 | 使用场景 |
|---|---|
| 隐藏按钮 | 普通用户不应看到管理入口 |
| 禁用按钮 + tooltip | 用户知道功能存在但当前无权限，如发布 |
| 403 页面 | 直接访问无权限路由 |

原则：业务工作台中的关键流程按钮优先禁用并说明原因；系统管理类入口优先隐藏。
