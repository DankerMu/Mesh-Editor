## 10. 前端界面设计

### 10.1 顶部：产品窗口选择区

```text
起报时间：YYYY-MM-DD HH
累计长度：24h / 48h / 168h
起始时效：+000h / +024h / +048h / ...
结束时效：自动计算
当前版本：原始 / 草稿 / 已保存版本
按钮：预览 / 应用 / 撤销 / 重做 / 保存 / 生成复盘 / 导出数据包
```

### 10.2 左侧：查看内容

```text
原始结果图
当前订正图
降水差异图
相态变化图
人工修改区域图
复盘组合图
```

首版不设置 850hPa、700hPa、500hPa 等交互图层开关。这些变量只进入复盘绘图任务。

### 10.3 中间：地图编辑区

```text
固定 PNG 底图
累计降水/相态色斑图
选区边界
笔刷范围
编辑预览效果
当前鼠标格点提示
```

### 10.4 右侧：编辑工具面板

分为四个页签：

```text
区域选择
降水调整
相态调整
操作历史
```

#### 区域选择

```text
多边形
绘制笔
笔刷
圆形，二期可选
清除选区
反选，二期可选
```

#### 降水调整

```text
定值：qpf = value
增加：qpf += delta
减少：qpf -= delta，最小为 0
乘系数：qpf *= factor
清零：qpf = 0，ptype = 0
弱降水筛查：qpf <= threshold 时清零
```

#### 相态调整

```text
设为无降水（ptype=0）
设为雨（ptype=1）
设为雪（ptype=2）
设为雨夹雪（ptype=3）
恢复原始相态
```

#### 操作历史

```text
操作时间
工具类型
变量
动作
影响格点数
影响面积
撤销/重做入口
```
