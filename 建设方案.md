# 固定区域 5km 累计降水与雨雪相态网格编辑及复盘系统建设方案

**版本**：V2.0  
**说明**：完整业务平台方案 + 工程开发设计索引  
**日期**：2026-05-15  
**适用范围**：固定区域 0.05° 等经纬度网格（业务称谓"5km 加密格点"）、24h/48h/168h 累计降水与相态产品订正、EC IFS 多要素复盘绘图  
**技术选型**：前端 Vue 3 + OpenLayers + TDesign，后端 Python FastAPI，部署 Docker  
**投影坐标系**：EPSG:4326（等经纬度投影）

---

## 方案概述

建设一个完整的固定区域网格编辑与复盘业务平台，覆盖数据接入、人工订正、版本审核、产品发布、复盘归档、历史分析和运维监控。系统只对累计降水和相态进行交互订正；EC IFS 的高空高度场、风、温度、湿度及其他变量不进入主编辑流程，而作为复盘绘图和后续分析的数据输入。

### 三个业务闭环

```
编辑闭环：
数据扫描 → 窗口生成 → QC 标记 → 选择窗口 → 编辑 → 预览/应用 → 保存版本

审核发布闭环：
保存草稿 → 提交审核 → 审核通过/退回 → 正式发布 → 归档

复盘闭环：
编辑结果 → 绘图任务队列 → 复盘组合图 → 复盘包归档 → 历史统计分析
```

### 关键参数

| 参数 | 值 |
|---|---|
| 网格尺寸 | 501 行 × 821 列 |
| 分辨率 | 0.05° × 0.05°（业务称谓"5km 加密格点"） |
| 地理范围 | 70-111°E, 25-50°N |
| 投影 | EPSG:4326（等经纬度） |
| 累计降水差分 | qpf = tp(end_lead) - tp(start_lead)，详见 §8.2.1 |
| 累计时段 | 24h / 48h / 168h |
| 时效步长 | 3h，000-240h |
| 起报时次 | 每日 08Z / 20Z |
| 相态编码 | 0=无降水, 1=雨, 2=雪, 3=雨夹雪 |

| 2.5-5 | (166,167,168) | 中灰 |
| 5-10 | (117,118,119) | 深灰 |
| ≥10 | (77,78,79) | 暗灰 |
